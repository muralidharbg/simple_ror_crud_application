    • Closure: Closures in javascript gives functions enclosed within other functions to have access to the variables set in the parent (outer) function.
    • I have used closures with asynchronous calls. The most recent situation I used it was for a page which had product listing. This product listing needs to be approved by the admin, for which each product has a button. I wrote an event listener for the button, which when clicked would trigger an asynchronous call to the server letting it know that the particular product clicked on needs to be approved.
    • You can find the snippet of code here: https://github.com/muralidharbg/se-team-7/blob/master/YouReview/WebContent/ApproveProducts.jsp
    • In the code you will find 
      product_listing_dom
      variable set in the parent function which hold a DOM element. Based on the server response the DOM element is then manipulated using the variable name.
